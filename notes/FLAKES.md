# NixOS on flakes

<https://nixos.wiki/wiki/Flakes>

## Upgrade nixpkgs

```console
nix flake update
nix build .#flake-name
```

Or in a single command.

```console
nix build .#flake-name --recreate-lock-file
```

## Update specific input flake

```console
nix flake lock --update-input neovim-flake
```
